# **App Name**: CareerCraft AI

## Core Features:

- AI Resume Restructuring: Master Resume Builder: Upload a resume (Word or PDF), and the AI will tool analyze and reformat it based on best practices. Assesses missing information and prompts the user with clarifying questions, ensuring a complete Master Resume.
- AI Job Spec Targeting: Targeted Resume Builder: Upload a job spec (Word, PDF, or URL), and the AI tool will tailor the Master Resume to highlight matching skills and experience. It assesses the fit and asks clarifying questions for better matching.
- Online Resume Generator: Online Resume: Generates a shareable online resume page, which users can embed or link to.
- LinkTree Bio Generator: LinkTree Generator: Generates a 'link in bio' style page.

## Style Guidelines:

- Primary color: HSL 210, 70%, 45% (RGB hex: #2289C3). A professional and trustworthy blue that evokes confidence and stability, suitable for career-related applications.
- Background color: HSL 210, 20%, 95% (RGB hex: #F0F4F7). Very light, desaturated blue. Provides a clean and modern backdrop that is easy on the eyes.
- Accent color: HSL 180, 60%, 40% (RGB hex: #1AA3A3). Provides a contrasting visual interest without being distracting.
- Body text and headline font: 'Inter', a sans-serif with a neutral, modern appearance. Good for long blocks of text, as expected in resumes.
- Code font: 'Source Code Pro' for displaying computer code.
- Simple, professional icons for resume sections (work experience, education, skills).
- Clean, single-column layout for readability and focus.
